# DESIGN.md — 上手吧 UI 设计标准（锁定版）

本文件为项目唯一设计标准，所有页面开发必须严格遵守。
源自两份基准文档合并：The Pristine Canvas 战略设计系统 + 微信小程序 UI 落地规范。

---

## 一、设计北极星

**"The Pristine Canvas"** — 不是清洁工具 App，是高端生活管家。
通过 Soft Minimalism（柔和极简）、留白驱动、编辑式排版，让界面像高端杂志，而不是功能堆砌的模板。

**品牌气质：** 安心（严谨排版）· 秩序（栅格对齐）· 克制（去装饰化、苹果式极简）

---

## 二、颜色系统（严格锁定）

### 允许使用的颜色

| 角色 | 色值 | 用途 |
|------|------|------|
| **主色** | `#F9E12A` | 核心操作、高亮状态、成功状态（代替绿色） |
| **主文字** | `#000000` / `#111111` | 标题、正文、强提示 |
| **辅助文字** | `#666666` | 说明文字、次要信息 |
| **弱文字** | `#999999` / `#888888` / `#8E8E93` | 提示、占位 |
| **失效文字** | `#CCCCCC` / `#AEAEB2` | 禁用态 |
| **主背景** | `#FFFFFF` | 页面主背景、卡片底 |
| **次背景** | `#F5F5F5` / `#f9f9f9` / `#f3f3f3` | 分区底色 |
| **深次背景** | `#e8e8e8` / `#e0e0e0` / `#EBEBEB` | 内嵌分组、进度条未到达 |
| **分割/边框** | `#EEEEEE` / rgba 低透明度 | 极细线（优先用留白代替） |

### 严禁使用的颜色

- **红色系**：`#FF3B30`、`#FF0000` 及任何红色 — 全部禁止
- **绿色系**：`#34C759`、`#1d7a35`、`#1a9e45` 及任何绿色 — 全部禁止
- **蓝色系**：`#007AFF` 及任何蓝色 — 全部禁止
- **紫色系**：全部禁止
- **橙色系**：`#f9a825`、`#FF9500`、`#d4a000` — 全部禁止
- **非标黄色**：`#f2c200`、`#7a6000`、`#6a5f00` — 非 `#F9E12A` 的黄色全部禁止
- **任何渐变色** — 全部禁止

### 语义色替代规则

| 语义 | 禁止用法 | 正确做法 |
|------|---------|---------|
| 成功/可用 | 绿色 | `#F9E12A` 黄色 + 黑色文字 |
| 错误/不可用 | 红色 | `#000000` 文字 + 下划线 + `#CCCCCC` 灰底 |
| 警告/待定 | 橙色 | `#F9E12A` 黄底 + `#000000` 文字 |
| 链接/信息 | 蓝色 | `#000000` 下划线文字 |
| 折扣/优惠 | 橙色/红色 | `#F9E12A` 黄底黑字标签，或 `#000000` 加粗文字 |

## 四、间距与圆角

**间距步进：** 8px 系统（8, 16, 24, 32, 48）
**页面侧边距：** 16px（32rpx）

| 元素 | 圆角 |
|------|------|
| 卡片 / 大按钮 | 12px（24rpx）|
| 标签 / 输入框 | 6px（12rpx）|
| 标签胶囊 | 4px（8rpx）|

## 五、表面与深度（The No-Line Rule）

### 禁止行为
- 禁止使用 `1px solid` 实线做分区（用色调过渡代替）
- 禁止厚重投影

### 表面层级
| 层 | 色值 | 用途 |
|-----|------|------|
| Base | `#f9f9f9` | 页面默认底 |
| Layer 01 浮动 | `#FFFFFF` | 主内容卡片 |
| Layer 02 内嵌 | `#e8e8e8` | 次级分组 |

### 投影规范
- 低浮动：`0 4px 20px rgba(0,0,0,0.05)`
- 原则：优先用细描边替代投影
- 如必须投影：Blur 32px / Color `#1a1c1c` at 4% / Y-Offset 8px

### 玻璃效果（Header/浮动栏）
- 背景：`surface` 色 80% 透明度
- 效果：`backdrop-filter: blur(20px)`

## 六、组件规范

### 按钮
| 类型 | 背景 | 文字 | 高度 | 圆角 |
|------|------|------|------|------|
| 主按钮 | `#F9E12A` | `#000000` Bold | 48px | 12px |
| 次按钮 | `#FFFFFF` | `#000000` + 1px 黑边 | 48px | 12px |
| 禁用 | `#F5F5F5` | `#CCCCCC` 无边 | 48px | 12px |
| 幽灵 | 透明 | `#000000` Medium | — | — |

**禁止：** 按钮文字超过 6 个字。

### 卡片
- 白底 + 极淡投影或细边框 + 12px 圆角
- 内部留白 16px
- 禁止卡片堆叠超过 3 层逻辑
- 禁止内容拥挤

### 标签
- 黑底白字 或 黄底黑字
- 高度 24px，圆角 4px

### 表单输入
- 扁平化：底部细线或浅灰背景块
- 高度不低于 52px
- 错误状态：黑色文字 + 下划线，严禁红色

### Header
- 纯白背景（或玻璃效果），文字居中
- 预留 44px 微信胶囊安全区

### Footer / 操作栏
- 固定底部，**不透明纯白背景**
- 高度 60-80px，顶部 0.5px 灰色分割线

### 空态
- 大面积留白
- 中心 48px 线性图标 + 14px 灰色文本
- 禁止使用通用图标，优先使用高对比度室内空镜

---

## 七、禁止事项（Critical）

1. **禁止颜色：** 严禁红、蓝、绿、紫、橙及非标黄
2. **禁止装饰：** 严禁渐变色、厚重投影、异形边框、花哨动效
3. **禁止做法：** 严禁按钮文字超 6 字、卡片堆叠超 3 层、divider 线分隔列表项（用 24px 留白）
4. **禁止图片：** 严禁假笑模特、低分辨率图、高饱和度滤镜图
5. **禁止表单：** 严禁红色错误提示

---

## 八、微信小程序约束

- 基准尺寸：375×812px
- 底部预留 Home Indicator 安全区（`safe-area-inset-bottom`）
- CSS 使用 Flex/Grid 布局
- 样式变量化：颜色和间距优先用 CSS 变量
- 黄色只用 `#F9E12A`，成功态用黄色代替绿色
